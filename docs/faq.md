## FAQ

**Who can see annotations?**

All logged-in users can view annotations for sites they have access to. Anonymous users cannot access this plugin.

**Who can edit or delete annotations?**

- Users can edit and delete their own annotations
- Users with admin access to a site can manage any annotation for that site
- Super users can manage all annotations across all sites

**How do I add an annotation?**

Go to **Administration > Personal > Annotations**, click the **Add Annotation** button, select a site, choose a date, type your note, and save.

**Can I export annotations?**

Yes. Click the **Export CSV** or **Export JSON** buttons in the toolbar to download all annotations you have access to.

**Does this plugin make external requests?**

No. All data stays within your instance. No external services are contacted.

**Is this compatible with the built-in Annotations feature?**

Yes. This plugin reads and writes the same annotation data as the built-in feature. It provides a more convenient admin interface for managing annotations in bulk.
